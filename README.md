# Real-time LED control with ESP32 + iOS (WS2812B)

Control your WS2812B LED strip **live from your phone** — no more uploading code just to tweak colours or animations.

## 🎯 Why this exists

If you've ever:
- Tweaked LED animations in code for ages
- Re-uploaded firmware just to change one effect
- Wanted faster iteration

👉 This solves that.

## ⚡ What it does

- 🎨 Paint individual LEDs in real-time
- 🌈 Control colours + built-in effects
- ⏱️ Create timed sequences
- 💾 Save and reuse designs
- 🔗 Works over BLE (low latency)

## 🧠 How it works

- ESP32 runs provided firmware
- iOS app sends real-time commands over BLE
- LEDs update instantly

## 🛠️ Hardware

- ESP32 (Dev Module)
- WS2812B LED strip (5V)
- (Optional) 330Ω resistor + 1000µF capacitor

## 🚀 Setup (2–5 mins)

1. Upload the firmware from this repo to your ESP32
2. Power your LED strip
3. Download the app:
👉 [App Store Link Here]

4. Connect via BLE and start painting LEDs instantly
